# ReelFire 最终验收清单

> 验收日期：2026-07-18。证据以 `TEST_REPORT.md`、`BUG_RECORD.md`、`INTEGRATION_TEST_REPORT.md` 和 `Acceptance_screenshot/` 为准。

## 1. 环境与启动

- [x] README 默认克隆最终 `main`，不依赖固定绝对路径；
- [x] `setup_environment.py` 可复用非 base Conda 环境或创建 Python 3.11 环境；
- [x] PyTorch 按 NVIDIA CUDA、macOS/MPS 或 CPU 自适应安装；
- [x] 官方 `yolo11n.pt` 自动下载并执行 SHA-256 校验；
- [x] FFmpeg/ffprobe 自动检查与安装；
- [x] `pip check` 返回 `No broken requirements found`；
- [x] `python app.py --host 127.0.0.1 --port 7880` 可启动；
- [x] `/api/health` 返回模型与 FFmpeg 就绪状态。

## 2. 上传、任务与异常

- [x] MP4 对战录像上传成功并立即返回唯一 `job_id`；
- [x] 任务按 `created -> queued -> running -> completed/failed` 真实变化；
- [x] 原始素材、`job.json`、报告、关键帧和结果按任务隔离；
- [x] 失败任务保留 `job.json` 与可读错误；
- [x] 运行中任务禁止删除，非法任务编号和路径穿越被拒绝；
- [x] 空文件、缺文件、不支持扩展名、伪装格式和损坏容器均有明确响应。

## 3. CV、评分与片段

- [x] OpenCV 正确读取时长、分辨率、FPS 和音轨；
- [x] YOLO11n 完成真实推理，保存真实类别、置信度和边界框；
- [x] 关键帧图片绘制实际检测框；
- [x] `object/scene_change/motion` 分数均在 `[0,1]`；
- [x] 综合分数使用 `0.45/0.35/0.20` 权重并写入报告；
- [x] 相邻高分帧按最小时间间隔去重；
- [x] 开头、结尾、目标时长过长和无效时长边界测试通过；
- [x] 候选片段满足 `0 <= start < end <= duration`；
- [x] 非法人工边界返回 400 与可读错误。

## 4. 页面与人工审核

- [x] 上传、加载、运行、完成和失败状态均有截图；
- [x] 结果页展示四项分数、权重、候选片段和关键帧；
- [x] 关键帧支持保留/忽略、排序、标签和备注；
- [x] 审核结果写回 JSON，刷新/历史重开后仍存在；
- [x] 完成与失败任务可从历史列表重新打开；
- [x] 项目名和关键帧备注恶意载荷按纯文本渲染，不执行存储型 XSS；
- [x] 浏览器完整回归无控制台错误、无失败请求。
